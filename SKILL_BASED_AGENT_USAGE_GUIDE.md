# Skill-Based Voice Agent Creation Guide

## Overview

The Skill-Based Voice Agent Framework allows you to create production-ready AI voice agents in **30-60 seconds** by selecting pretrained skills instead of writing complex system prompts.

## How to Select Agent Skills

### Step 1: Access the Agent Creation Dialog

1. Navigate to the **Virtual Agents** page
2. Click the **"Create New Agent"** button
3. You'll see two tabs:
   - **Skill-Based (Recommended)** - Quick creation with pretrained skills
   - **Manual Configuration** - Full control for advanced users

### Step 2: Choose a Pretrained Skill

The skill selection interface displays available skills organized by category:

#### Available Skill Categories:

1. **Content & Asset Distribution**
   - Whitepaper Distribution Agent
   - Research Report Promotion Agent
   - eBook Promotion Agent (coming soon)
   - Case Study Distribution Agent (coming soon)

2. **Event Promotion**
   - Webinar Registration Agent
   - Executive Dinner Invitation Agent
   - Leadership Forum Invitation Agent (coming soon)

3. **Survey & Research**
   - Market Research Survey Agent (coming soon)
   - Customer Feedback Agent (coming soon)

4. **Qualification & Appointment**
   - Appointment Setting Agent
   - MQL Qualification Agent (coming soon)
   - BANT Qualification Agent (coming soon)

#### How to Select a Skill:

1. **Browse the skill cards** - Each card shows:
   - Skill name and description
   - Category badge
   - User-facing value proposition
   - Checkmark when selected

2. **Click on a skill card** to select it
3. The skill's input form will appear below

### Step 3: Configure the Skill

Each skill has **Required** and **Optional** inputs:

#### Example: Whitepaper Distribution Agent

**Required Inputs:**
- **Whitepaper PDF** - Upload your asset file
- **Whitepaper Title** - e.g., "The Future of B2B Marketing"
- **Publishing Organization** - e.g., "Acme Corporation"

**Optional Inputs (Enhance Personalization):**
- **Target Persona** - Marketing Leaders, Sales Leaders, IT Leaders, etc.
- **Key Topics to Highlight** - Comma-separated topics

#### Example: Appointment Setting Agent

**Required Inputs:**
- **Meeting Purpose** - Discovery Call, Product Demo, Strategy Consultation
- **Meeting Duration** - 15, 30, 45, or 60 minutes
- **Calendar Booking Link** - Your Calendly/Chili Piper link

**Optional Inputs:**
- **Qualification Criteria** - BANT framework criteria
- **Meeting With** - Name/title of person prospects will meet

### Step 4: Configure Voice Settings

Choose your agent's voice characteristics:

1. **Voice Provider**: OpenAI or ElevenLabs
2. **Voice**: Nova (Female), Alloy (Neutral), Echo (Male), Shimmer (Female)

### Step 5: Preview the Compiled Prompt (Optional)

Before deploying, you can preview the complete system prompt that will guide your agent:

1. Click **"Preview Prompt"** button
2. Review the compiled prompt showing:
   - **Sources Used**: Universal Brain, Skill Intelligence, Org Intelligence
   - **Full System Prompt**: The complete instruction set
   - **Word Count & Token Estimate**
3. Use the copy button to save the prompt if needed

The preview shows how the framework layers:
- **Layer 0**: Vision & Mission Governance (highest priority)
- **Layer 1**: Universal Voice Agent Brain (compliance, etiquette, objection handling)
- **Layer 2**: Skill-Specific Intelligence (call flows, tactics, success metrics)
- **Layer 3**: Organization Context (your company info, ICP, positioning)
- **Layer 4**: User Inputs (the values you provided)

### Step 6: Deploy Your Agent

Click the **"Deploy Agent"** button to create your agent instantly.

The agent is immediately ready to:
- Make outbound calls
- Follow pretrained call flows
- Handle objections professionally
- Comply with GDPR/TCPA regulations
- Track success metrics automatically

## Understanding Skill Intelligence

### What's Inside a Skill?

Each skill contains **1,500-2,000 words** of pretrained intelligence:

#### 1. Call Flow Stages
Step-by-step progression through the conversation:
- Stage 1: Introduction (with time limits)
- Stage 2: Value delivery (persona-adapted)
- Stage 3: Interest gauge (conversational)
- Stage 4: Permission-based next step
- Stage 5: Outcome classification

#### 2. Objection Response Library
Pretrained responses for common objections:
- "I'm too busy"
- "Not interested"
- "Send me information"
- "Already have a solution"
- "How did you get my number?"
- "What's the catch?"

#### 3. Persona Adaptation
Different messaging for different roles:
- **Marketing Leaders** → Strategy, ROI, customer insights
- **Sales Leaders** → Productivity, pipeline, revenue
- **IT Leaders** → Technology, integration, security
- **Operations Leaders** → Efficiency, process, scalability
- **C-Suite** → Business outcomes, competitive advantage

#### 4. Success Metrics
Built-in performance tracking:
- **Whitepaper Agent**: 60%+ email permission, 40%+ downloads
- **Webinar Agent**: 35%+ registration, 60%+ live attendance
- **Appointment Agent**: 20-30% booking rate, 70-80% show-up rate

#### 5. Compliance Safeguards
Automatic adherence to:
- GDPR/TCPA regulations
- Opt-out handling
- Time-based calling restrictions (8am-6pm)
- Permission-based follow-ups
- Honest positioning (no misleading language)

## Mission-Aligned Execution

### Layer 0: Vision & Mission Governance

All agents are governed by your organization's Vision & Mission **before** executing any tactics.

#### How It Works:

Every agent action passes through these decision filter questions:

1. **Mission Alignment:** Does this action advance our stated mission?
2. **Vision Consistency:** Does this reinforce our long-term vision?
3. **Value Preservation:** Does this build trust and credibility?
4. **Relationship Impact:** Does this optimize for long-term relationship value?

#### Conflict Resolution:

If a skill tactic conflicts with your Vision/Mission, the agent will:
- **Modify** the tactic to align
- **Suppress** the tactic entirely
- **Abort** the interaction if alignment is impossible

Example:
- **Skill Tactic:** "Push for immediate commitment"
- **Vision:** "Build long-term, trust-based relationships"
- **Agent Action:** Modifies approach to be consultative instead of pushy

#### Mission-Alignment Tracking:

Each call is flagged with:
- **Aligned** - All actions consistent with Vision/Mission
- **Neutral** - No conflicts detected
- **Modified** - Adjusted tactics to align with Vision/Mission
- **Aborted** - Stopped interaction due to mission conflict

This ensures your agents are **extensions of your organization's intent**, not just automation.

## Troubleshooting

### "Missing Inputs" Error
**Problem:** You didn't fill in all required fields
**Solution:** Check for fields marked with a red asterisk (*) and fill them in

### "Invalid Input Format" Error
**Problem:** Input doesn't match expected format (e.g., URL, file type)
**Solution:** Ensure URLs start with `https://` and files are the correct type (.pdf for documents)

### Preview Shows "Organization Intelligence Not Found"
**Problem:** No Organization Intelligence configured
**Solution:** Go to AI Studio → Organization Intelligence and create your org profile

### Agent Not Following Skill Logic
**Problem:** Manual edits overriding skill intelligence
**Solution:** Recreate the agent using skill-based creation without manual edits

## API Integration (For Developers)

### Fetch Available Skills
```javascript
GET /api/virtual-agents/agent-skills

Response: [
  {
    id: 'whitepaper_distribution',
    name: 'Whitepaper Distribution Agent',
    category: 'content_distribution',
    description: '...',
    requiredInputs: [...],
    optionalInputs: [...]
  }
]
```

### Preview Compiled Prompt
```javascript
POST /api/virtual-agents/agent-skills/preview

Body: {
  agentName: 'Sarah Chen',
  skillId: 'whitepaper_distribution',
  skillInputValues: {
    asset_file: 'future-of-b2b-marketing.pdf',
    asset_title: 'The Future of B2B Marketing Automation',
    publishing_org: 'Acme Corporation'
  }
}

Response: {
  preview: '# AGENT IDENTITY\n\nYou are Sarah Chen...',
  wordCount: 3500,
  estimatedTokens: 4550
}
```

### Create Skill-Based Agent
```javascript
POST /api/virtual-agents/create-from-skill

Body: {
  agentName: 'Sarah Chen',
  skillId: 'whitepaper_distribution',
  skillInputValues: { ... },
  voice: 'nova',
  provider: 'openai'
}

Response: {
  agent: { id: '...', name: 'Sarah Chen', ... },
  skillMetadata: {
    skillName: 'Whitepaper Distribution Agent',
    category: 'content_distribution',
    successMetrics: [...],
    callFlowStages: [...]
  },
  sources: [
    'Universal Voice Agent Brain',
    'Skill: Whitepaper Distribution Agent',
    'Organization Intelligence'
  ]
}
```
