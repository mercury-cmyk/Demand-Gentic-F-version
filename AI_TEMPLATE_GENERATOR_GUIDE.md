1. Go to **Left Sidebar** → **Aura AI** section

### 15. **Troubleshooting**

**Q: Template not generating?**
- Check your internet connection
- Try a more specific description
- Use keywords like "webinar" or "whitepaper"

**Q: Subject line too generic?**
- Customize after generation
- Use recipient name + benefit
- Add urgency indicators (limited time, exclusive)

**Q: Brand colors not applying?**
- Select brand palette from right sidebar
- Click "Generate Email" again
- Regenerate after changing palette

**Q: CTA not working?**
- Verify URL is valid (https://...)
- Test in preview before sending
- Use dynamic URL variables if needed
